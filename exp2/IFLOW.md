# IFLOW.md - 二叉树与哈夫曼编码项目

## 项目概述

这是一个关于数据结构的编程项目，主要涉及二叉树（Binary Tree）的实现以及哈夫曼编码（Huffman Coding）算法的构建。项目要求基于马丁·路德·金的演讲《I have a dream》构建哈夫曼编码树，并对特定单词进行编码。

项目当前包含一个二叉树的基本实现（bt1.cpp）和一个描述项目要求的PDF文档（exp2.pdf）。

## 项目结构


- `bt1.cpp` - 二叉树（BT）和二叉树节点（BTNode）的C++实现，包含基本的树遍历、树操作功能
- `exp2.pdf` - 项目详细要求文档，描述了需要实现的四个主要部分
- `exp2.txt` - 空文本文件（可能是用于输入测试数据）
- `IFLOW.md` - 本项目上下文文件

## 当前实现功能

### 二叉树（BT）类功能
- 构建先序序列二叉树
- 递归遍历：前序、中序、后序遍历
- 层序遍历
- 计算树高
- 计算叶子节点数量
- 镜像翻转树
- 内存管理（析构函数）

### 项目要求的四个部分（待实现）
1. 实现二叉树数据结构 BinTree
2. 基于 BinTree 构建 HuffTree
3. 借助位图类 Bitmap 定义 Huffman 二进制编码串类型 HuffCode
4. 实现 Huffman 编码算法

## 技术栈

- C++ 11 或更高版本（使用了 `#include<bits/stdc++.h>`、`vector`、`queue`、`istringstream` 等）
- 使用面向对象编程方法实现数据结构

## 代码规范

- 类的实现遵循标准C++面向对象设计模式
- 使用智能指针或手动内存管理（当前实现使用手动new/delete）
- 使用const引用传递避免不必要的拷贝

## 构建和运行

由于项目目前只有一个C++源文件，可以通过以下方式编译运行：

```bash
g++ -o bt1 bt1.cpp
./bt1
```

程序会提示用户输入先序序列，使用 # 表示空节点，空格分隔。
